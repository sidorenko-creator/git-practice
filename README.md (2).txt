## 🛠️ Featured Projects

### Project 1: BetterImages AI Web Application (`betterimages.ai`)

*   **Role:** Sole QA Engineer
*   **Scope:** Full-cycle functional, usability, UI/UX, and regression testing across 11 core web pages.
*   **Environment:** Cross-device testing using **BrowserStack** on real devices, including iPhone 16 Pro Max and Google Pixel 9 Pro.

#### 🎯 Key Achievements & Responsibilities:

*   **Vulnerability & Bug Discovery:** Identified and documented **25 defects**, including **10 high-severity** issues (Stored XSS vulnerability, broken Google/Apple OAuth flow, and email enumeration). Each bug report was delivered with structured reproduction steps, screenshots, and remediation notes.
*   **Documentation & Traceability:** Designed a **10-page Test Coverage Matrix** from scratch to handle regression scope, requirement traceability, and fix verification.
*   **Cross-Device UI/UX Consistency:** Verified iOS/Android layouts, responsive navigation, and core user flows prior to production release. Delivered a final QA Sign-Off report featuring UX, performance, and security recommendations.
